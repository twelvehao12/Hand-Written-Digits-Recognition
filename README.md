# Hand-Written-Digits-Recognition
基于pytorch和opencv的手写数字识别

源代码在对应的brunch里

---

使用PySide2开发Python的图形界面，使用MFC开发C++的图形界面

模型选用ResNet18

使用Pytorch训练模型，并导出为onnx格式，使用OpenCV加载推理

---

CUDA: 11.7

Python: 1.9.13(Pytorch); 3.8.16(PySide2)

Pytorch: 2.0.1

onnx: 1.13.1

onnxruntime: 1.12.1

OpenCV-python: 4.7.0

OpenCV(MSVC): 4.6.0
